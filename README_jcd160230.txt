Binary file reader and parser for records.

To run, simply issue the following commands:

git clone https://github.com/JamesD777/program6.git
cd program6
make
./cdk

The project includes simple algorithmic logic linked with the given cdk example, the hardest part being manipulating the data to display what we want to the matrix. Runtimes depend on how long you display the given matrix. To close the matrix after running, simple press any key to close and stop the program!
